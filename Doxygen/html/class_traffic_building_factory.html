Detailed Description

Factory class for creating TrafficBuilding instances.

This class inherits from BuildingFactory and provides a static method to create TrafficBuilding objects with specified types and capacities, initializing them with an initial state.

static TrafficBuilding * TrafficBuildingFactory::createTrafficBuilding ( TransportType type)
inlinestatic

Creates a TrafficBuilding instance.

This method initializes a TrafficBuilding with the specified transport type and capacity. The building is set to an initial state of 'UnderConstruction' and costs are deducted from the resource manager's budget.

Parameters
typeThe transport type for the TrafficBuilding (e.g., TAXI, BUS, TRAIN, AIRPORT).
Returns
TrafficBuilding* A pointer to the newly created TrafficBuilding instance.
