\doxysubsection{Detailed Description}
Manages traffic and transport types within a city. 

The \doxylink{class_transportation}{Transportation} class handles the registration and management of different types of traffic buildings. It calculates traffic flow based on the capacities of these buildings, updates transport capacities, and provides information about the current transport state.
