\doxysubsection{Detailed Description}
Represents a building that provides transportation services. 

This class inherits from \doxylink{class_building}{Building} and represents a building dedicated to transportation. It includes a transport type, cost, and methods for providing services, retrieving transport type and cost, and cloning the building instance.
